# stack

Коллекция конвертированных Jupyter-ноутбуков (ipynb → markdown) по анализу биржевых данных и машинному обучению.

## Описание

Репозиторий содержит **62 файлов** — результаты исследований, экспериментов и учебных задач в формате Markdown (сконвертировано из Jupyter Notebook). Темы:

### Биржевой анализ
- Точки минимальных выплат (опционы)
- Статистика по дням недели, месяцам (RTS, MIX, Si)
- Объёмы в кластерах (5-минутные бары)
- Buy & Hold стратегия
- Загрузка данных с MOEX ISS

### Машинное обучение
- LSTM-нейросети
- Decision Tree Regression
- Нейросети на Keras (Boston dataset)
- Сохранение/загрузка моделей

### Прочее
- Работа с Google Disk API
- Календарные расчёты

## Формат

Каждый `.md` файл — самостоятельное исследование с кодом, пояснениями и картинками (в папках `*_files/`).
